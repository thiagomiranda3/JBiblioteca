JBiblioteca
=========

JBiblioteca é uma aplicação em Java que utiliza o Swing Builder do NetBeans que eu desenvolvi para treinar. Ele simula o comportamento de um gerenciador de biblioteca, mas esta estupidamente simples ainda. Procurei colocar o máximo de validação possível para que o programa ficasse legal, mas ainda tem bastante coisa pra fazer.
